# 파워리프팅 약점 구간(Sticking Point) 극복 가이드

**스티킹 포인트를 찾는 팁(A note on finding sticking points):** 일반적으로 진짜 약점은 스티킹 포인트보다 실제로는 살짝 아래에 위치해 있습니다. 스티킹 포인트는 단순히 약점 구간에서 바벨이 감속(deceleration)된 결과로 나타나는 현상입니다.

## 2. 벤치 프레스 (Bench Press)

**참고:** RAW 리프터를 위한 기준입니다. 대부분의 사람들이 중간 지점(mid point)을 어깨의 약점으로 생각하지만, 어깨를 훈련하면 삼두근도 함께 훈련되기 때문에 실용적인 측면에서 이 둘을 굳이 분리할 필요는 없습니다.

| 구분 | 가슴 구간 (Off The Chest) | 중간 ~ 락아웃 구간 (Mid To Lockout) |
| :--- | :--- | :--- |
| **구간 강화** | • 정지 프레스 (2-5초)<br>• 와이드-그립 정지 프레스 | • 스포토 프레스<br>• 2-5 보드 프레스<br>• 핀 프레스 |
| **잠재적 약점** | **흉근 (Pecs)** | **삼두근/어깨 (Triceps/Shoulders)** |
| **약점 근육 강화** | • 와이드 그립 벤치 프레스<br>• 폼 보드 프레스<br>• 와이드 스포토 프레스<br>• 인클라인 프레스<br>• 체스트 딥스<br>• 체스트 플라이 (케이블 또는 머신)<br>• 프레스 변형 (덤벨 활용) | • 클로즈 그립 벤치 프레스<br>• 2-5 보드 프레스<br>• 클로즈 그립 보드 프레스<br>• 핀 프레스<br>• JM 프레스<br>• 스포토 프레스<br>• 오버헤드 프레스<br>• 플로어 프레스<br>• 아놀드 프레스<br>• 덤벨 숄더 프레스 (스탠딩 & 시티드)<br>• 슬링샷<br>• 삼두 익스텐션 변형 (케이블, 로프, 덤벨, 밴드, 오버헤드, 바벨 등) |
| **폭발력 훈련** | • 스피드 벤치 (@60%)<br>• 밴드 벤치 (@50% 무게 + 25-35% 밴드 장력) | • 스피드 벤치 (@60%)<br>• 밴드 벤치 (@50% 무게 + 25-35% 밴드 장력)<br>• 슬링샷 |

## 3. 스쿼트 (Squat)

| 구분 | 최하단 구간 (The Hole) | 무릎 모임 (Knee Cave) | 엉덩이가 먼저 뜸 (Hips Rising First) |
| :--- | :--- | :--- | :--- |
| **구간 강화** | • 정지 스쿼트<br>• 로우 박스 스쿼트 (변형: 와이드, 내로우)<br>• 핀 스쿼트<br>• 앤더슨 스쿼트 | • 와이드 스탠스 스쿼트<br>• 와이드 박스 스쿼트<br>• 와이드 정지 스쿼트 | • 프론트 스쿼트<br>• 내로우 스쿼트<br>• 내로우 박스 스쿼트 |
| **잠재적 약점** | **다양함 / 테크닉 (Various/Technique)** | **둔근/고관절 (Glutes/Hips)** | **대퇴사두 (Quads)** |
| **약점 근육 강화** | • 정지 스쿼트<br>• 로우 박스 스쿼트 (변형: 와이드, 내로우)<br>• 핀 스쿼트<br>• 앤더슨 스쿼트<br>• 최하단 구간에서의 테크닉 집중/안정감 | • 와이드 스탠스 스쿼트<br>• 와이드 박스 스쿼트<br>• 스트레이트 레그 데드리프트<br>• 루마니안 데드리프트<br>• 힙 쓰러스트<br>• 바벨 글루트 브릿지<br>• 와이드 박스 스쿼트<br>• 불가리안 스플릿 스쿼트<br>• 런지/스텝백 런지<br>• 와이드 레그 프레스<br>• 글루트 햄 레이즈<br>• 리버스 하이퍼익스텐션<br>• 풀스루<br>• 레그 컬 (시티드, 라잉, 싱글 레그) | • 프론트 스쿼트<br>• 클로즈 스탠스 하이바 스쿼트<br>• 레그 익스텐션<br>• 불가리안 스플릿 스쿼트<br>• 런지/스텝백 런지<br>• 핵 스쿼트<br>• 바벨 핵 스쿼트<br>• 내로우 레그 프레스<br>• 로우 박스 스쿼트<br>• 핀 스쿼트<br>• 썰매 끌기 (Sled Pulls) |
| **폭발력 훈련** | • 스피드 스쿼트 (@60%)<br>• 밴드 스쿼트 (@50% 무게 + 25-35% 밴드 장력)<br>• 박스 점프/중량 박스 점프<br>• 박스 스쿼트 (변형: 내로우, 와이드) | • 스피드 스쿼트 (@60%)<br>• 밴드 스쿼트 (@50% 무게 + 25-35% 밴드 장력)<br>• 박스 점프/중량 박스 점프<br>• 와이드 박스 스쿼트 | • 스피드 스쿼트 (@60%)<br>• 밴드 스쿼트 (@50% 무게 + 25-35% 밴드 장력)<br>• 박스 점프/중량 박스 점프<br>• 내로우 박스 스쿼트 |
