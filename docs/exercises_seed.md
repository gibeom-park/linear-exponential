## 제외 (현 시점에서 시드에 미포함)

다음 운동들은 출처 문서에 등장하지만 분류가 모호하거나 powerlifting 보조에서 벗어나 제외했음. Open Question 으로 보관.

- **도구류**: 슬링샷
- **컨디셔닝/폭발력 점프**: 박스 점프, 중량 박스 점프, 썰매 끌기

추후 카테고리(`kind = tool` / `kind = conditioning`) 를 추가할 때 다시 포함 검토.
